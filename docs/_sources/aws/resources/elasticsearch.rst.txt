.. _aws.elasticsearch.filters.json-diff:

json-diff
+++++++++
Compute the diff from the current resource to a previous version.

A resource matches the filter if a diff exists between the current
resource and the selected revision.

Utilizes config as a resource revision database.

Revisions can be selected by date, against the previous version, and
against a locked version (requires use of is-locked filter).

.. container:: toggle

  

  .. raw:: html
     
    <div class="header docutils container" style=""></div>

  .. code-block:: yaml

    properties:
      selector:
        enum:
        - previous
        - date
        - locked
      selector_value:
        type: string
      type:
        enum:
        - json-diff
    required:
    - type

.. _aws.elasticsearch.filters.kms-key:

kms-key
+++++++
Filter a resource by its associcated kms key and optionally the aliasname
of the kms key by using 'c7n:AliasName'

:example:

.. code-block:: yaml

    policies:
      - name: elasticsearch-kms-key
        resource: aws.elasticsearch
        filters:
          - type: kms-key
            key: c7n:AliasName
            value: "^(alias/aws/es)"
            op: regex
