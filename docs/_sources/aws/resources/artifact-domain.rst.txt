.. _aws.artifact-domain.actions.delete:

delete
++++++
:example:

Delete empty domains older than 30 days.

.. code-block:: yaml

  policies:
    - name: empty-delete
      resource: artifact-domain
      filters:
         - type: value
           key: createdTime
           value_type: age
           op: greater-than
           value: 30
         - assetSizeBytes: 0
      actions:
         - delete
